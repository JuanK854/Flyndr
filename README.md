## Flyndr - Flight Search UI

This is a simple **Next.js 14** flight search experience using the **App Router** and **Tailwind CSS**.

- Single-page flight search with mock data only
- Dark, modern UI with responsive grid layout
- No authentication, database, or real API calls

### Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
